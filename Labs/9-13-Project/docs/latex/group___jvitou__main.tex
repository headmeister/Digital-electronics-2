\doxysubsection{Detailed Description}
D\+AC signal generating, using A\+VR M\+Cu and R2R D\+AC with output on 16x2 L\+CD and U\+A\+RT. 

This project uses simple R2R D\+AC to generate some usefull waveforms in frequency range from 100 Hz to 2k\+Hz. It uses two main timers, first to sample the output signal and second to check pressed buttons and update signal parameters and display them onto L\+CD or send throught U\+A\+RT.

Uses Peter Fleurys libraries for L\+CD and U\+A\+RT communication
