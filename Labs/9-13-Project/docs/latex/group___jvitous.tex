\doxysubsection{Detailed Description}
Generate signal based on entered type frequency and other parameters. 


\begin{DoxyCode}{0}
\DoxyCodeLine{\textcolor{preprocessor}{\#include <\mbox{\hyperlink{_sig__gen_8h}{Sig\_gen.h}}>} }
\end{DoxyCode}


This library generates signals from L\+U\+Ts or using formulas. The available waveforms are\+: Sine, positive ramp, negative ramp, triangle, square, E\+CG or D\+T\+MF with parameter. The framebuffer is filled with corresponding data which can then be output to the output P\+O\+R\+TX. The default output precision is of data type uint8\+\_\+t ~\newline


\begin{DoxyNote}{Note}
Uses Peter Fleurys libraries for L\+CD and U\+A\+RT communication 
\end{DoxyNote}
